我自己使用的一个工具箱插件，提供多个实用功能增强思源笔记的使用体验。

## 功能列表

### 1. 图片压缩

自动或手动压缩笔记中的图片，有效减小笔记体积，支持多种压缩策略和智能跳过机制。

#### 使用方式

1. **自动压缩**：在设置中开启"自动压缩图片"后，粘贴或上传图片时自动进行压缩
2. **批量压缩**：点击工具栏图标 → 压缩当前笔记图片
3. **单张压缩**：右键点击图片 → 压缩图片

#### 压缩模式

- **普通压缩**：使用固定的质量参数（0.1-1.0）压缩图片
- **智能压缩**：基于 MS-SSIM 相似度评估，自动调整压缩参数以达到目标相似度（默认 99%）

#### 跳过压缩的图片类型

为保护特定资源或避免无效压缩，以下情况会自动跳过压缩：

1. **Excalidraw 插件资源**
   - 文件名或路径包含 `excalidraw-` 的图片
   - 原因：Excalidraw 插件的嵌入资源被压缩后会导致插件功能异常
   - 提示：`跳过 Excalidraw 插件资源: {文件名}`

2. **WebP 格式图片**（可配置）
   - 已是 WebP 格式的图片
   - 原因：避免重复压缩造成质量损失
   - 可在设置中关闭此选项

3. **小文件图片**（可配置）
   - 小于指定大小的图片（默认 100KB）
   - 原因：小文件压缩收益不大，避免不必要的处理
   - 可在设置中调整最小文件大小阈值

4. **视频文件**
   - 支持 `.mp4`、`.avi`、`.mov`、`.wmv`、`.flv`、`.webm`、`.mkv`、`.m4v`、`.3gp`、`.ogv` 等格式
   - 原因：视频文件不支持图片压缩
   - 提示：`视频文件不支持压缩`

#### 设置选项

- **自动压缩图片**：开启后粘贴/上传图片时自动压缩
- **图片压缩质量**：0.1-1.0，数值越小文件越小但质量越低
- **跳过 WebP 格式**：是否跳过已是 WebP 格式的图片
- **跳过小文件**：是否跳过小于指定大小的图片
- **最小文件大小**：设置跳过压缩的文件大小阈值（字节）
- **启用智能压缩**：使用 MS-SSIM 相似度评估进行智能压缩
- **目标相似度**：智能压缩的目标相似度百分比（1-100）

### 2. 标签排序

智能排序标签列表，常用的标签会自动排在前面，提高标签检索效率。

#### 功能特点

- 自动记录每个标签的使用次数
- 使用次数越多，排序越靠前
- 支持反转排序（在设置中开启）
- 相关数据存储在 `SiYuan/data/storage/petal/toolkit-plugin-siyuan/tagSort.json`

### 3. 冲突对比

当笔记出现冲突时（如多端同步冲突），可视化对比两个版本的差异。

#### 使用方式

1. 点击工具栏图标 → 冲突对比
2. 插件会自动在打开的冲突文档中对比两个版本
3. 新增的内容会用红色边框高亮显示
4. 支持同步滚动两个版本（鼠标滚轮操作）

#### 适用场景

- 多端同步产生的冲突文档
- 需要对比两个版本差异的场景
- 快速定位新增或修改的内容
